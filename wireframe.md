# 미술심리상담 NALDA — 랜딩페이지 와이어프레임

> 레퍼런스: https://arttherapynalda.com/
> 데스크탑 기준(최대 콘텐츠 폭 약 1200px), 반응형 고려.

---

## 1. Header (고정 상단 바)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ [🦋 NALDA 로고]   회사소개  미술심리상담  그룹 프로그램  강의  쇼핑몰        │
│                                          [🔍 검색어를 입력하세요]  🛒(0) 마이페이지 | 로그아웃 │
└─────────────────────────────────────────────────────────────────────────────┘
```
- 좌측: 로고(나비 + "미술심리상담 / NALDA")
- 중앙: GNB 메뉴 5개 (회사소개 / 미술심리상담 / 그룹 프로그램 / 강의 / 쇼핑몰)
- 우측: 검색 인풋(돋보기), 장바구니 아이콘(배지 숫자), 마이페이지, 로그아웃

## 2. Hero Slider (메인 배너 슬라이드)

```
┌───────────┬───────────────────────────────┬───────────┐
│ slide 좌  │      현재 배너 (중앙 강조)      │  slide 우 │
│ (peek)    │   ◀  [배너 이미지]  ▶          │  (peek)   │
└───────────┴───────────────────────────────┴───────────┘
                         ● 2 / 4
```
- 배너 4장 캐러셀(좌우 peek 형태, 중앙 활성 슬라이드 확대)
- 좌우 화살표, 페이지 인디케이터(2/4)
- 각 배너는 링크 보유(강의/쇼핑몰/강사소개/프로그램)

## 3. Notice Bar (공지사항)

```
┌─────────────────────────────────────────────────────────────┐
│ [공지] 공지사항2                                          ⌄  │
└─────────────────────────────────────────────────────────────┘
```
- "공지" 뱃지 + 최신 공지 제목, 우측 펼침(아코디언) 화살표

## 4. Quick Menu (아이콘 8개)

```
 ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐
 │ 🟣 │ │ 🟡 │ │ 🟢 │ │ 🔴 │ │ 🟢 │ │ 🟢 │ │ 🟡 │ │ 🔴 │
 └────┘ └────┘ └────┘ └────┘ └────┘ └────┘ └────┘ └────┘
 미술심리 그룹    강의   쇼핑몰  마이    상담    수강    주문
 상담     프로그램              페이지  내역    내역    내역
```

- 가로 8개 정렬(모바일은 4×2)

## 5. 프로그램 목록 (가로 카드 캐러셀)

```
 🏷 프로그램 목록                                        [전체보기 ▸]
 ◀ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ▶
   │ img  │ │ img  │ │ img  │ │ img  │
   │ 제목 │ │ 제목 │ │ 제목 │ │ 제목 │
   │가격~ │ │가격~ │ │가격~ │ │가격~ │
   └──────┘ └──────┘ └──────┘ └──────┘
```
- 섹션 제목(아이콘 + "프로그램 목록") + 우측 "전체보기" 버튼(보라 pill)
- 카드: 썸네일 / 제목 / "00,000원~"
- 좌우 화살표로 캐러셀 이동
